Open-source software tools developed by BCG

We put particular emphasis on software quality and the adherence of our tools with coding quality standards by using the SoftWipe tool developed in our sister lab at the Heidelberg Institute for Theoretical Studies. The three fancy tools presented at the bottom of this page used to evidently be placeholders for the real tools we are developing, they have not been removed for fun's sake.

bigrig software (under development)

bigrig is a program to simulate historical biogeographical regions under the so-called DEC(+J) model.

sintax-ng software (under development)

sintax-ng is a complete re-implementation and improvement (regarding scalability, adaptability) written in RUST of Robert Edgar's sintax software which is a simple non-Bayesian taxonomy classifier for 16S and ITS sequences.

P = NP software

Our new tool for exactly solving NP-hard problems in polynomial time.

The MCMC convergence tool

A provably correct tool that tells you if your MCMC chain has converged.

Machine learning meta-tool

Test this new meta tool for machine learning that given any kind of data will tell you what your hypothesis is.
